Référence de syntaxe de l'anglais

Page 1

  • Sujet + Verbe + Complément : "She sings a song."
  • Sujet + Verbe + Complément + Complément d'Objet : "He gave her a gift."
  • Sujet + Verbe + Complément + Complément de Lieu : "We live in a city."
  • Sujet + Verbe + Complément + Complément de Temps : "They arrived on Monday."

Page 2

  • Phrase Négative : Sujet + Auxiliaire + "not" + Verbe : "She does not like coffee."
  • Phrase Interrogative (Yes/No) : Auxiliaire + Sujet + Verbe : "Do you play guitar?"
  • Phrase Interrogative (Question ouverte) : Mot interrogatif + Auxiliaire + Sujet + Verbe : "What did she say?"
  • Phrase Conditionnelle : If + Clause conditionnelle + Clause principale : "If it rains, we will stay at home."

Page 3

  • Phrase Passive : Sujet + Auxiliaire + Verbe à la forme passive + par + Agent (optionnel) : "The book was written by him."
  • Phrase Impérative : Verbe à l'impératif (+ Sujet optionnel) : "Sit down."
  • Phrase Exclamative : Quel/Quelle + Nom + Verbe + ! : "Quelle belle journée !"
  • Phrase Comparative : Sujet + Verbe + plus/moins/aussi + adjectif + que : "She is taller than him."
